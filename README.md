### 목적
채굴, 코인거래 기능을 직접 구현해봄으로써 비트코인 블록체인 이해하기

## 실행 방법

### 1. 모듈 설치
```bash
yarn
cd client && yarn
cd ..
```

### 2. 서버 열기
```bash
yarn run dev
```
```bash
yarn run dev-peer
```
```bash
yarn run dev-peer2
```
